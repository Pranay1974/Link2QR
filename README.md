## 🚀 Overview

**QR Studio** is a sleek, modern, and production-ready QR code generator that allows users to instantly convert any text or URL into high-resolution QR codes.

Designed with a **clean UI**, fast performance, and **download capability**, it’s perfect for personal, business, and branding use.

---

## ✨ Features

* 🔗 Convert any **URL or text → QR Code**
* 📏 Multiple resolution options:

  * 200x200 (Standard)
  * 400x400 (Medium)
  * 600x600 (HD)
  * 1000x1000 (Print Quality)
* ⚡ Instant QR generation
* 💾 Download QR as PNG
* 🎨 Modern dark UI with smooth UX
* 🔒 Uses secure external API (QRServer)

## ⚙️ How It Works

1. User enters text or URL
2. Selects QR resolution
3. App calls QRServer API
4. Displays QR instantly
5. User can download it
